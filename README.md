# GoodGolf Draw Next.js Prototype

This is a Next.js, mobile-first website prototype for a subscription golf score draw platform with charity contributions.

Run it with:

```bash
npm run dev
```

Then open the local Next.js URL shown in the terminal.

## Included

- Home page
- Login page
- Signup page
- Logged-in player dashboard
- Monthly/yearly subscription selector
- Charity selection with minimum contribution messaging
- Golf score entry with:
  - score range validation from 1 to 45
  - required date
  - duplicate date prevention
  - latest five scores only
- Monthly draw simulation with 5, 4, and 3 match prize tiers

## Asset

The hero image was generated with the built-in image generation tool and copied into `public/assets/hero-impact.png` for Next.js use.
